# Advanced PHP: Set 2
## PHP Internals, Memory Management & Execution Lifecycle

## 📘 10 Conceptual Questions
1. **The Zval Structure:** What exactly is a `zval` (Zend Value) in PHP's engine? Explain its components (type, value, `refcount`, and `is_ref`) and how PHP uses this structure to manage variables efficiently under the hood.
2. **Copy-on-Write (CoW) Optimization:** How does PHP optimize memory when you assign `$a = $b`? Explain the mechanism of Copy-on-Write, and precisely what action (e.g., using references `&`, or modifying the variable) breaks this optimization and forces a memory duplication.
3. **Garbage Collection (GC) & Cyclic References:** Simple reference counting fails when objects reference each other (circular references). How does PHP’s Cycle Collector (introduced in PHP 5.3) detect and clean up these "roots" without causing performance degradation on every request?
4. **OpCache Mechanics:** What does OPcache *actually* cache? Explain the difference between caching source code vs. caching compiled Zend Opcodes, and why `opcache.validate_timestamps=0` is mandatory for production performance.
5. **JIT (Just-In-Time) Compiler Realities:** How does PHP 8’s Tracing JIT work? Why does it provide massive speedups for CPU-bound tasks (math, image processing) but often yields negligible or even negative performance gains for typical I/O-bound web requests (database queries, API calls)?
6. **The Danger of References (`&`):** Why is passing variables by reference generally considered an anti-pattern in modern PHP? Explain how it permanently disables Copy-on-Write for that variable, leading to unexpected memory bloat and harder-to-reason-about state.
7. **Generators (`yield`) & Lazy Evaluation:** How does the `yield` keyword transform a function into a Generator object? Explain how this allows you to process millions of rows with $O(1)$ memory complexity, compared to the $O(N)$ memory explosion of returning a massive array.
8. **Long-Running Process Memory Leaks:** Why do Laravel Queue Workers or CLI scripts slowly consume memory over time? Explain common culprits (e.g., accumulating event listeners, unclosed database cursors, static variables, ORM model caching) and how to mitigate them.
9. **String Interning:** How does PHP optimize memory for identical string literals? Explain the difference between request-level string interning and *permanent* string interning (which requires OPcache and CLI/FPM persistence).
10. **Object Destruction (`__destruct`):** When exactly is the `__destruct` magic method called? What are the dangers of performing heavy I/O, database queries, or throwing exceptions inside a destructor, especially during script shutdown?
