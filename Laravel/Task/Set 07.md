# Laravel Modern Stack & Advanced Patterns: 10 Questions & 10 Tasks

## 📘 10 Conceptual Questions
1. **Laravel Precognition & Partial Validation:** How does Precognition enable real-time validation without full form submission? Explain the `Precognition` header, partial validation flow, and how it differs from traditional AJAX validation in terms of security and performance.
2. **Database Generated/Virtual Columns:** How do generated columns (`->generatedAs()`) work in Laravel migrations? When should you offload computation to the database vs PHP, and how do they impact indexing, write performance, and query caching?
3. **Laravel Pulse & Custom Telemetry:** What metrics does Pulse track natively? How do you extend it with custom recorders (e.g., business events, queue depth, cache hit ratios), and when should Pulse replace vs complement traditional APM tools?
4. **API Standards: JSON:API vs REST vs GraphQL:** How does Laravel implement each? Compare `laravel-json-api` resource transformers, standard REST with API Resources, and Lighthouse GraphQL. When does strict API compliance reduce frontend complexity vs create backend overhead?
5. **Task Scheduling & Mutex Internals:** How does `->withoutOverlapping()` work under the hood? Explain cache-based mutexes, timezone resolution in the scheduler, `->runInBackground()`, and how to safely handle missed executions or server downtime.
6. **Security Headers & CSP Architecture:** How do you configure `Strict-Transport-Security`, `Content-Security-Policy`, `X-Content-Type-Options`, and `Referrer-Policy` in Laravel? Explain how CSP mitigates XSS, how to safely whitelist inline scripts, and why nonce/hash-based CSP is preferred over `unsafe-inline`.
7. **Pest PHP vs PHPUnit Architecture:** How does Pest's expectation API, `test()` closures, and dataset-driven testing change test maintainability? Explain parallel testing isolation, snapshot testing, and how to safely migrate legacy PHPUnit suites without breaking CI pipelines.
8. **File-Based Routing (Folio) vs Traditional Routing:** How does Folio change the request lifecycle? Explain per-page middleware, layout inheritance, route parameter extraction from filenames, and when file-based routing simplifies vs complicates large-scale applications.
9. **Database Connection Limits & Pooling:** How does Laravel manage DB connections across web requests and queue workers? Explain persistent connections, connection poolers (PgBouncer/ProxySQL), the "max_connections" bottleneck, and how to prevent connection exhaustion in high-concurrency scenarios.
10. **Zero-Downtime Deployment & Rollback Mechanics:** How do tools like Envoyer/Forge achieve zero-downtime? Explain symlink swapping, asset versioning/fingerprinting, queue draining, maintenance mode bypass for deployers, and how to implement safe database migration rollbacks.
