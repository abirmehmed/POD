# Laravel Production & Scale: 10 Questions & 10 Tasks

## 📘 10 Conceptual Questions
1. **Multi-tenancy Architectures:** Compare single-database (shared schema + `tenant_id`), multi-database (schema-per-tenant), and hybrid approaches. When does each make sense, and how does Laravel handle tenant routing/database switching?
2. **Laravel Scout & Search Indexing:** How does Scout bridge Eloquent models to search drivers (Algolia, Meilisearch, database)? Explain the implications of keeping search indexes in sync with DB changes, and how to handle partial updates or nested relationships.
3. **Horizon & Queue Orchestration:** How does Horizon improve upon basic `queue:work`? Explain tags, metrics, autoscaling workers, balance strategies, and how to configure supervisor/process managers for production resilience.
4. **Socialite & OAuth2 Flow:** How does Socialite abstract provider-specific OAuth flows? Explain the difference between stateful (web) and stateless (API) authentication, how to handle token refreshes, and security implications of storing provider tokens.
5. **Laravel Dusk & Browser Testing:** How does Dusk differ from standard HTTP testing? Explain ChromeDriver requirements, handling asynchronous UI, debugging failed tests, and why it's essential for complex checkout flows or SPAs.
6. **Schema Dumps & Migration Squashing:** Why does Laravel offer schema dumps? Explain how `schema:dump` and `schema:load` speed up CI/test environments, and when squashing migrations becomes dangerous in team workflows.
7. **Feature Flags & A/B Testing:** How do you implement feature toggles without `if (env('FEATURE_X'))`? Explain database-backed flags, percentage rollouts, user-segment targeting, and how to avoid flag pollution in the codebase.
8. **Health Checks & Uptime Monitoring:** How do you create a `/health` endpoint that verifies DB, cache, queue, and disk connectivity? Explain how to integrate with monitoring tools and handle false positives without alert fatigue.
9. **CI/CD Pipeline Design:** What should a Laravel GitHub Actions/GitLab CI pipeline include? Explain steps for linting, testing, asset compilation, cache warming, and zero-downtime deployment strategies.
10. **Performance Profiling Strategy:** How do Clockwork, Laravel Debugbar, and Blackfire differ in profiling? Explain how to identify bottlenecks in N+1 queries, view rendering, and memory leaks without impacting production performance.
