# Laravel API Architecture, Microservices & Security Compliance: 10 Questions & 10 Tasks

## 📘 10 Conceptual Questions
1. **API Versioning & Deprecation Cycles:** How do you evolve an API without breaking existing clients? Compare URL versioning (`/v1/`, `/v2/`), header negotiation, and response shaping. How do you implement soft deprecation, sunset headers, and migration windows safely?
2. **Sanctum vs Passport vs Raw JWT:** When should you use Laravel Sanctum (SPA/token) vs Passport (OAuth2) vs custom JWT? Explain stateless vs stateful auth, token refresh flows, revocation strategies, and security trade-offs for mobile vs web consumers.
3. **Outbox Pattern & Eventual Consistency:** How do you guarantee message delivery when publishing domain events to a message broker or microservice? Explain the database outbox table, transactional writes, background publishers, and idempotent consumers.
4. **Sliding Window Rate Limiting & Abuse Detection:** How does a sliding window algorithm differ from fixed window or token bucket? How do you implement per-user/IP limits with Redis, detect bot patterns, and apply progressive penalties without blocking legitimate traffic?
5. **RFC 7807 Problem Details for API Errors:** Why should APIs return structured error responses instead of HTML or generic JSON? How do you map Laravel exceptions, validation failures, and business rules to `type`, `title`, `status`, `detail`, and `instance` fields consistently?
6. **CORS, CSP & API Security Headers:** How do you configure Cross-Origin Resource Sharing for SPAs/mobile apps without opening security holes? Explain `Access-Control-*` headers, `Strict-Transport-Security`, `X-Content-Type-Options`, and how to validate them with security scanners.
7. **OpenAPI/Swagger Auto-Generation & Contract Validation:** How do you keep API documentation in sync with code? Explain schema-first vs code-first approaches, Laravel package integrations, request/response validation against specs, and breaking-change detection in CI.
8. **HTTP Caching, ETags & Conditional Requests:** How do you reduce server load and improve perceived performance? Explain `Cache-Control`, `ETag`, `Last-Modified`, `If-None-Match`, and stale-while-revalidate strategies for read-heavy endpoints.
9. **Structured Logging & Request Tracing:** How do you make logs queryable and actionable in production? Explain JSON logging, correlation IDs, log levels, PII redaction, and how to trace a request across web → queue → external API → database.
10. **GDPR/CCPA Compliance in Laravel:** How do you implement data subject rights (export, delete, restrict processing) without breaking referential integrity or audit trails? Explain PII encryption, anonymization strategies, consent tracking, and retention policies.
