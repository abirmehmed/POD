# JavaScript Programming Language: Set 2
## Advanced Mechanics: Metaprogramming, Concurrency & Performance

## 📘 10 Conceptual Questions

1. **Generators & Iterators:** What is a generator function (declared with `function*`), and how does the `yield` keyword pause and resume execution? How does the Iterator protocol (`[Symbol.iterator]`) work under the hood, and how does it power modern syntax like `for...of` loops and the spread operator (`...`)?
2. **Proxies & Reflect:** What is a `Proxy` object, and how does it intercept fundamental operations (like `get`, `set`, or `apply`)? How does the `Reflect` API complement Proxies, and what are practical use cases for them (e.g., data validation, creating reactive state like in Vue 3, or hiding properties)?
3. **Memory Management & V8 Garbage Collection:** How does the V8 engine handle memory allocation and Garbage Collection (e.g., Mark-and-Sweep, Generational GC)? What are the most common causes of memory leaks in JavaScript (e.g., forgotten timers, detached DOM nodes, closures holding large scopes), and how do you identify them?
4. **WeakMap & WeakSet:** How do `WeakMap` and `WeakSet` differ from standard Maps and Sets regarding garbage collection? Why are their keys strictly required to be objects, and how are they used to implement private data or manage DOM node metadata without causing memory leaks?
5. **Web Workers & True Parallelism:** Since JavaScript is single-threaded, how do Web Workers achieve true parallel execution? What is the performance cost of communicating between the main thread and a worker (structured cloning), and how does `SharedArrayBuffer` bypass this for high-performance scenarios?
6. **ES6 Classes & Private Fields:** How does the modern `class` syntax differ from traditional prototype-based inheritance under the hood? Explain the ES2022 private class fields (`#fieldName`) and private methods. How do they provide true encapsulation compared to the old "closure-based" or `Symbol`-based privacy hacks?
7. **TypedArrays & Binary Data:** How does JavaScript handle raw binary data? Explain the relationship between `ArrayBuffer` (the raw data), `DataView` (the reader/writer), and TypedArrays (like `Uint8Array` or `Float32Array`). Why are these crucial for WebGL, WebSockets, and file manipulation?
8. **Currying, Partial Application & Composition:** What is the difference between currying (transforming `f(a,b,c)` to `f(a)(b)(c)`) and partial application? How do these functional programming concepts enable function composition (`pipe` / `compose`) to build complex logic from small, pure, reusable functions?
9. **Event Delegation & Propagation:** How does the DOM event propagation model work (Capture Phase ➔ Target Phase ➔ Bubble Phase)? How does Event Delegation leverage bubbling to handle events for dynamically added elements efficiently, and when would you use `e.stopPropagation()`?
10. **Design Patterns in JS:** How are classic software patterns adapted for JavaScript? Explain how the Module pattern (using closures), the Observer pattern (Event Emitters/Pub-Sub), and the Proxy pattern (for lazy loading or virtual properties) are implemented in modern JS architecture.
