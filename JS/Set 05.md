# JavaScript Programming Language: Set 5
## Engine Internals, WebAssembly, and Advanced Browser APIs

## 📘 10 Conceptual Questions

1. **V8 Engine & JIT Compilation:** How does the V8 engine (used in Chrome and Node.js) compile JavaScript to machine code? Explain the roles of the **Ignition** interpreter and the **TurboFan** optimizing compiler. What are "Hidden Classes" (Shapes) and "Inline Caching," and how do they make object property access incredibly fast?
2. **Async Iterators & `for await...of`:** How do Async Iterators differ from standard synchronous Iterators? How does the `for await...of` loop handle backpressure and sequential asynchronous data streams (e.g., reading chunks from a database, paginated API calls, or large files)?
3. **WebAssembly (Wasm) & JS Interop:** What is WebAssembly, and how does it differ from JavaScript in terms of execution speed and memory management? How do you pass complex data (like strings or arrays) between JS and Wasm, given that Wasm's linear memory only understands raw numbers and bytes?
4. **Advanced Type Coercion & The Abstract Equality Algorithm:** Why does the infamous `[] == ![]` evaluate to `true` in JavaScript? Break down the ECMAScript Abstract Equality Comparison Algorithm. When should you strictly use `===`, and are there *any* valid, safe use cases for `==`?
5. **Intersection Observer & Lazy Loading:** How does the `IntersectionObserver` API work under the hood, and why is it vastly superior to listening to `scroll` events and using `getBoundingClientRect()`? How do you use it to implement infinite scrolling or native-like image lazy loading?
6. **The View Transitions API:** What is the new View Transitions API? How does it allow for smooth, cross-document (Multi-Page Apps) and same-document (Single-Page Apps) animations without relying on complex, heavy JavaScript animation libraries like Framer Motion or GSAP?
7. **Decorators (Stage 3) & Metaprogramming:** How do the newly standardized JavaScript Decorators work? How can they be used to intercept method calls, add metadata, or implement cross-cutting concerns (like logging, authentication checks, or memoization) without cluttering the core business logic?
8. **Web Crypto API & SubtleCrypto:** How does the native `crypto.subtle` API handle encryption, hashing, and key generation? Why are these cryptographic operations inherently asynchronous, and how do you securely encode strings to bytes (using `TextEncoder`) before hashing them?
9. **AST (Abstract Syntax Tree) Manipulation:** What is an AST, and how do foundational tools like Babel, ESLint, Prettier, and TypeScript use it? How can you write a simple script to parse JS code into an AST, traverse it, and programmatically transform or restrict specific syntax?
10. **Micro-Frontends & Module Federation:** In massive enterprise applications, how does the Micro-Frontend architecture work? How do tools like Webpack Module Federation allow multiple independent teams to deploy separate React/Vue/Vanilla apps that share dependencies at runtime and run as a single cohesive unit?
